The curriculum heavily emphasises the importance of clean code. Projects

UniBoard

iOS Programming • Spring 2017

For my final app I created a social media application called Uniboard. Uniboard is an image sharing application for university members. It was built through swift and database handling is done through firebase. The current functionalities of the app allow users to create accounts, follow and unfollow other members, view and like other member’s pictures and upload pictures to your feed.
